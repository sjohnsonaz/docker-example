``` bash
az aks get-credentials --resource-group [[Resource Group]] --name [[Cluster Name]]
```


# Kubernetes

## Get Kubernetes Context

``` bash
kubectl config current-context
```


## Set Kubernetes Context

``` bash
kubectl config use-context [[Kubernetes Context]]
```
